1 # Description
2 PRAgMaTIc (Parallel anisotRopic Adaptive Mesh ToolkIt) provides 2D/3D
3 anisotropic mesh adaptivity for meshes of simplexes. The target
4 applications are finite element and finite volume methods although
5 it can also be used as a lossy compression algorithm for 2 and 3D data
6 (e.g. image compression). It takes as its input the mesh and a metric
7 tensor field which encodes desired mesh element size
8 anisotropically.
9 
10 The toolkit is written in C++ but also provides interfaces for Fortran. It
11 has been integrated with [FEniCS/Dolfin](http://fenicsproject.org) and
12 integration with PETSc/DMPlex is planned. One of the design goals of PRAgMaTIc
13 is to develop highly scalable algorithms for clusters of multi-core and
14 many-core nodes. PRAgMaTIc uses OpenMP for thread parallelism and MPI for
15 domain decomposition parallelisation.
